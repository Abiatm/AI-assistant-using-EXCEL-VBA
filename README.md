# AI-assistant-using-EXCEL-VBA
For this analysis in Excel, I integrated an AI assistant built with VBA, embedding a LLaMA model to enable natural language querying of the dataset. This setup made it possible to automatically generate and visualize charts beyond the original dashboard, uncovering deeper insights and improving overall data understanding.
I explored the customer dataset featured in Freedom Oboh YouTube video, which also inspired the dashboard design for this analysis. Here are some of the key insights that emerged:
Out of every 500 customers, 11.8% belong to the informal sector, highlighting the need for stronger awareness and financial inclusion campaigns.
The dataset covers the period 2015 to 2025, with 2017 recording the highest number of customers.
The average customer, aged 46 years, earns $114.4K.
